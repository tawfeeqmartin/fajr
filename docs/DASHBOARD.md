# Fajr Accuracy Dashboard

An interactive HTML dashboard visualizing the autoresearch experiment trajectory.

## Current Metrics (as of Experiment 6)
- WMAE: 1.55 minutes across 18 cities
- 93.6% reduction from baseline (24.17 min)
- All per-prayer MAEs under 2 minutes

The dashboard includes:
- WMAE trajectory across experiments
- Per-prayer radar chart
- Baseline vs current comparison
- Per-city accuracy table
- Experiment timeline with classifications

See `fajr-dashboard.html` for the interactive version.
